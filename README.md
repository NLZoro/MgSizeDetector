üìñ Description
---------------
The AI Cup Sizing Tool is a static website designed to help users measure their cup size using artificial intelligence and computer vision. This project utilizes the MediaPipe Pose library to detect the user's body and calculate their cup size. The tool is easy to use, accurate, and provides a fun experience for users.

The AI Cup Sizing Tool is built using HTML, CSS, and JavaScript, making it a lightweight and efficient solution for users. The website is optimized for mobile devices, allowing users to access the tool from anywhere. With its simple and intuitive interface, users can quickly and easily measure their cup size and get recommendations for the perfect fit.

The project's goal is to provide a convenient and accurate way for users to measure their cup size, eliminating the need for physical measurements or guessing. By using AI and computer vision, the tool can detect the user's body and calculate their cup size with high accuracy. The project's mission is to make bra shopping easier and more enjoyable for users, and to provide a valuable resource for users who struggle with finding the perfect fit.
